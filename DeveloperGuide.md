# Zanata Developer Guide

# Development tools

- Java Development Kit (JDK) 1.6 - yum install java-1.6.0-openjdk-devel
- GIT - Source code repository
- Eclipse IDE for J2EE (Helios/3.6)
- MySQL database
- JBoss AS
- Maven v2.X - v3
